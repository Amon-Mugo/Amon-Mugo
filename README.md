Hi 👋 My name is Amon Kariuki
=============================

Data Engineer
-------------

I'm a data engineer specialising in production-grade pipelines — dbt models, data warehousing, and orchestration on GCP. My background is in machine learning, so I bring an ML-aware lens to the pipelines I build: designing data models with downstream feature engineering and model training in mind, not just BI dashboards. Currently building cloud-native ELT pipelines (GCS → BigQuery → dbt) orchestrated with Airflow, deployed via CI/CD with Workload Identity Federation.

* ✉️  You can contact me at [amonkariuki325@gmail.com](mailto:amonkariuki325@gmail.com)
